# 기사 품질 스펙 — 시스템 요구사항

> 작성일: 2026-04-22
> 기준: 약업신문 + 뉴시스 레퍼런스 기사 분석 기반

---

## 목표

"기사처럼 보이게"가 아니라 **핵심 이슈 1개를 숫자+사례+맥락으로 압축한 전문지 기사체.**

---

## 통과 기준

### A. 첫 문단 (리드)

- 2문장 이내
- 핵심 사실 1개
- 숫자 1개 이상
- 왜 중요한지 1개

### B. 본문

- 대표 사례 1개 이상
- 비교 프레임 1개 이상 ("기존 대비", "전년 동기 대비" 등)
- **결측/디버그 문구 0개** (데이터 부족, 확인 불가, 분석 불가 금지)
- **내부 지시문 0개** (약제팀은 ~하라, 즉시 점검하라 금지)

### C. 마무리

- 과장 금지
- "전망이다" 남발 금지
- 향후 관전 포인트 1개만

---

## 금지 표현 (하드 룰)

| 금지 | 대안 |
|------|------|
| "데이터 부족" | 안 씀 |
| "확인 불가" | "세부 일정은 공개되지 않았다" |
| "분석 불가" | 해당 소스 기사에서 제외 |
| "약제팀은 ~하라" | 안 씀 |
| "즉시 점검하라" | 안 씀 |
| "RA/MA는" | 안 씀 |
| "그래서 뭐?" | 안 씀 |
| "FACT DATA" | 안 씀 |
| "[FACT DATA]" | 안 씀 |

---

## 이상적 구조

```
1. 리드: 핵심 사실 + 숫자 (2문장)
2. 대표 사례: 가장 강한 예시 (1~2단락)
3. 맥락/비교: 기존 대비 뭐가 달라졌는지 (1단락)
4. 확장 의미: 시장/제도/개발 영향 (1단락)
5. 마무리: 전망 + 단서 (1~2문장)
```

## 좋은 문장 vs 나쁜 문장

### 좋음
> 일본 후생노동성이 엔허투의 HER2 양성 고형암 적응증을 확대하면서, 다암종 정밀의료 전략이 다시 주목받고 있다.

### 나쁨
> PMDA는 엔허투를 승인했다. 향후 관련 변화가 주목된다.

### 좋음
> 코크란은 7종의 항체치료제, 총 17개 임상시험(참여자 2만342명) 데이터를 분석한 결과, 아밀로이드 제거가 인지기능 개선으로 이어지지 않는다고 결론 내렸다.

### 나쁨
> 연구 결과가 발표됐다. 데이터를 분석했다. 효과가 없는 것으로 나타났다.

## 레퍼런스 기사 패턴

### 약업신문 (수치 나열형)
- 리드: 팩트+수치
- 본문: 사례→성공률→비용절감→제조 확장
- 마무리: 전망+주의사항
- 인용: 간접만
- 수치 밀도: 높음

### 뉴시스 (논란/영향형)
- 리드: 임팩트+감성
- 본문: 출처→결과→반론→역사→전환점→국내기업
- 마무리: 직접 인용으로 종결
- 수치 밀도: 중간

### 공통
- 실무 지시 없음
- 결측 노출 없음
- 보도체 ("~로 나타났다", "~전망이다")
- 국내 맥락 연결
