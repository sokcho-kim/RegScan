## 1. 실험 개요

### 1.1 목표
MFDS 허가 의약품(44,035건)에 대해 HIRA 급여 정보 연결

### 1.2 매칭 전략
```
MFDS ITEM_SEQ → 약가마스터 품목기준코드 → 일반명코드 → HIRA ingredient_code
                                                    ↓
                                              급여기준, 상한가
```

### 1.3 사용 데이터

| 데이터 | 건수 | 출처 |
|--------|------|------|
| MFDS 허가정보 | 44,035 | 공공데이터포털 |
| 약가마스터 | 305,522 | data.go.kr |
| ATC 매핑 | 21,953 | data.go.kr |
| HIRA 적용약가 | 59,004 | biz.hira.or.kr |

## 3. 분석

### 3.1 매칭 실패 원인

**1. 약가마스터 일반명코드 미채움 (55%)**
```
약가마스터 전체: 305,522건
일반명코드 있음: 136,310건 (44.6%)
일반명코드 없음: 169,212건 (55.4%)
```

**2. 코드 커버리지 제한**
```
약가마스터 고유 일반명코드: 13,476개
HIRA 고유 의성분코드: 9,002개
교집합: 6,068개 (67.4%)
```

**3. INN Fallback 제한**
- ATC 매핑 INN 수: 1,382개
- 신약(semaglutide 등) 미등재

### 3.2 "미연결" 16,919건 분석

| 원인 | 예상 비율 |
|------|----------|
| 약가마스터에 품목기준코드 없음 | 35% |
| 일반명코드 미채움 | 40% |
| 일반명코드 → HIRA 미매칭 | 20% |
| 기타 | 5% |

### 3.3 "삭제" 9,769건 의미

- 과거에 급여 등재되었으나 현재 삭제된 품목
- 제조 중단, 급여 삭제, 코드 변경 등
- 참고용으로만 활용

## 4. 결론

### 4.1 성과

1. **61.6% 매칭 성공**: MFDS 44,035건 중 27,116건 HIRA 연결
2. **핵심 약물 검증**: pembrolizumab, nivolumab 등 8개 중 7개 급여 확인
3. **Exact Match 유지**: Fuzzy matching 없이 코드 기반 정확 매칭

### 4.2 한계

1. 약가마스터 일반명코드 55% 비어있음
2. semaglutide 등 신약 미매칭
3. 최대 커버리지 67.4%로 제한

### 4.3 개선 방안

| 방안 | 예상 효과 | 우선순위 |
|------|----------|----------|
| 제품코드(보험코드) 직접 매칭 | +10% | 높음 |
| 약가마스터 일반명코드 보완 | +15% | 중간 |
| 품목명 보조 매칭 | +5% | 낮음 (신뢰도 주의) |
