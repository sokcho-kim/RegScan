### 2.5 ClinicalTrials.gov API v2

| 항목 | 내용 |
|------|------|
| **출처** | ClinicalTrials.gov |
| **API** | `clinicaltrials.gov/api/v2/studies` |
| **수집방식** | REST API (무료, 키 불필요) |
| **갱신주기** | 실시간 |
| **라이선스** | Public Domain |
| **로컬 캐시** | 없음 (런타임 조회) |
| **핵심필드** | `resultsSection` (primary outcome, adverse events, HR, CI, p-value) |
| **코드 위치** | `regscan/ingest/clinicaltrials.py`, `regscan/parse/clinicaltrials_parser.py` |
| **조회 전략** | Phase 3(결과 있음) → Phase 3(결과 없음) → Phase 2(결과 있음) 순 |

**파이프라인 연동:**
- `publish_articles.py`에서 기사 생성 시 약물별 CT.gov 조회
- `_fetch_ctgov_results_batch()` → 배치 검색
- `parse_results_section()` → HR, CI, p-value, 이상반응, 한계점 자동 추출
- LLM 입력에 `clinical_trial_results` 필드로 주입

### 2.6 의약품주성분 마스터 (브릿지 핵심)

| 항목 | 내용 |
|------|------|
| **출처** | 공공데이터포털 (HIRA) |
| **URL** | https://www.data.go.kr/data/15067461/fileData.do |
| **수집방식** | 파일 다운로드 |
| **갱신주기** | 연간 (~10월) |
| **라이선스** | 공공데이터 |
| **로컬 파일** | `data/bridge/yakga_ingredient_master.csv` |
| **건수** | 59,631건 (20,114 유니크 일반명코드) |
| **HIRA 커버리지** | 99.6% (9,002 HIRA 코드 중 8,962개 매칭) |
| **코드 위치** | `regscan/map/ingredient_bridge.py` |

**매칭 파이프라인:**
```
MFDS INN → normalize → 의약품주성분 마스터 exact match → 일반명코드
  → HIRA 적용약가 ingredient_code → 급여여부 + 상한가
```

**매칭 성공률** (v6 실험 기준):
- 정규화 매칭: 74.8% (32,940/44,035)
- ATC fallback: 0.6% (254건)
- 미매칭: 24.3% — 대부분 한약재(31%), 비급여(36%), 마스터 미등록(34%)

## 3. 데이터 흐름

```
┌────────────────────────────────────────────────────────────┐
│                 글로벌 규제 데이터                            │
│  FDA openFDA API  ─→  INN + 승인일                          │
│  EMA Open Data    ─→  INN + 승인일 + 적응증 + therapeutic_area│
└──────────────────────────┬─────────────────────────────────┘
                           │ INN 정규화
                           ▼
┌────────────────────────────────────────────────────────────┐
│                 브릿지 (INN → 성분코드)                       │
│  의약품주성분 마스터  ─→  일반명코드 (exact match, 74.8%)     │
│  ATC 코드 매핑        ─→  fallback (+0.6%)                   │
└──────────────────────────┬─────────────────────────────────┘
                           │ ingredient_code
                           ▼
┌────────────────────────────────────────────────────────────┐
│                 국내 데이터                                   │
│  MFDS 허가정보       ─→  허가 여부 + 브랜드명                 │
│  HIRA 적용약가       ─→  급여 여부 + 상한가 (4,527 성분코드)  │
│  ClinicalTrials.gov  ─→  임상 결과 (HR, PFS, 이상반응)       │
│  CRIS (참고)         ─→  국내 임상 건수 (건수만, 상업 금지)    │
└──────────────────────────┬─────────────────────────────────┘
                           │
                           ▼
┌────────────────────────────────────────────────────────────┐
│              기사 생성 파이프라인                              │
│  DomesticImpactAnalyzer → LLMBriefingGenerator (V4)         │
│    ├─ Python 팩트: d_day, 승인표, 비용 시나리오               │
│    ├─ LLM 인사이트: headline, key_points, 3섹션              │
│    └─ HTML 렌더링: Jinja2 + Tailwind                         │
└────────────────────────────────────────────────────────────┘
```
