#### 3. MFDS 안전성 서한/회수 수집기

국내 식약처 안전성 사각지대 해소 (기존 EMA safety만 수집).

| 파일 | 변경 |
|------|------|
| `regscan/ingest/mfds_safety.py` | **신규** |

**이슈 해결 (4/20)**:
- 회수 API: 활용신청 완료
- 안전성 서한: API 미제공 확인 → **Playwright 크롤러 → httpx+bs4로 최종 전환**
  - `nedrug.mfds.go.kr/pbp/CCBAC01` SSR 페이지라 Playwright 불필요
  - **TLS 1.2 강제 필요**: nedrug.mfds.go.kr가 TLS 1.3 미지원, httpx 기본 설정으로 `ConnectError` 발생 → `ssl.create_default_context()` + `maximum_version=TLSv1_2`로 해결
  - 테스트 통과: 2건 수집 (안전성 속보 배포 알림, 설글리코타이드 정보 서한)
