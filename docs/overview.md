# RegScan 프로젝트 개요

> 최종 수정: 2026-01-29

---

## RegScan이란?

**RegScan**은 의료·제약·보험 영역에서
**글로벌 규제(FDA, EMA 등)와 학술 근거의 변화를 '스캔'하여
한국 보험·심사·급여 관점의 영향으로 번역하는
Regulatory Intelligence Engine**이다.

RegScan은 뉴스를 요약하지 않는다.
보이지 않는 규제 구조와 근거의 흐름을
**X-ray / MRI처럼 투과적으로 분석**한다.

---

## MedClaim과의 관계

```
┌─────────────────────────────────────────────────────┐
│  MedClaim (본체)                                    │
│  - 인터랙티브 Q&A                                   │
│  - Citation / Split View                            │
│  - Hallucination 통제                               │
│  - "정답을 말해주는 시스템"                          │
├─────────────────────────────────────────────────────┤
│  RegScan (서브 프로젝트)          ◀── 현재 위치     │
│  - 정보 흐름 생성기                                  │
│  - 변화 감지 → 카드/피드/리포트 콘텐츠               │
│  - 개인화 추천의 재료 공급자                         │
│  - "보여줄 정보를 만드는 엔진"                       │
└─────────────────────────────────────────────────────┘
```

### RegScan의 역할 (명확히)

> **RegScan은 답을 하지 않는다.**
> **RegScan은 메인화면에 흘릴 정보를 만든다.**

| 하는 것 | 하지 않는 것 |
|---------|-------------|
| Signal Feed 생성 | Chat 응답 |
| Citation 메타데이터 | Citation UI/정책 |
| 변화 감지 + 카드 콘텐츠 | Hallucination 통제 |
| 개인화 태깅 | 답변 정책 |

---

## 프로젝트 목적

1. **언론 기사 중심 정보 소비 구조 탈피**
   - 뉴스 기사 X → 1차 소스(기관 데이터)만 사용
   - 저작권 문제 없는 공공 데이터 활용

2. **메인화면 콘텐츠 풍부화**
   - 빈 검색창 → 정보가 흐르는 포털
   - 사용자가 질문하지 않아도 볼거리 제공

3. **유료 가치 창출**
   - 공공 데이터 = "무료" 인식 극복
   - 큐레이션 + 개인화로 차별화

---

## 핵심 철학

### ❌ 하지 않는 것
- 언론 기사 요약
- 논문 전문 재배포
- 단순 RAG 검색 서비스

### ⭕ 하는 것
- 규제 변화 **신호(Signal)** 감지
- 임상·학술 **근거 흐름(Evidence Trend)** 분석
- 제도 간 **영향 매핑(Impact Mapping)**
- 사용자 맥락 기반 **개인화 분석**

---

## 처리 파이프라인

```
Ingest → Parse → Scan → Map → Personalize → Report
```

| 단계 | 설명 | 현재 상태 |
|------|------|----------|
| **Ingest** | 1차 데이터 수집 | 🔨 구현 중 |
| **Parse** | 문서 구조화 | ⬜ 대기 |
| **Scan** | 변화·신호 감지 | 🔨 구현 중 |
| **Map** | 글로벌 → 한국 제도 번역 | ⬜ Phase 2 |
| **Personalize** | 역할·기관 개인화 | ⬜ Phase 2 |
| **Report** | 리포트 생성 | ⬜ Phase 2 |

## RegScan의 정의 (한 문장)

> **RegScan은 규제와 근거의 변화를 '읽는' 것이 아니라
> 의료·제약·보험 의사결정에 미칠 영향을 '촬영'하는 시스템이다.**
